Analysis of eBay Roman coins¶

There are a lot of coins sold on eBay. But what are the most popular. I would have said Gordian III antoninianus, but turns out it is Hadrian denarius.

Textmining¶

I textmined the titles for emperor and denomination.

Emperors¶

Luckly, someone make a nice table of emperors. Constantine the great is also known as Constantine I or Constantine Senior.

Denominations¶

Denominations came from wikipedia. Plus a few that wiki seems to not know, such as aurelianus (a post-Aurelian reform XXI antoninianus) and minims. Unfortunately, I did not want to waste time with Latin/English plurals. 1 as, 2 asses. 1 quadrans, 2 quadrantes.

One thing I could do, that I haven't is classify by gold, silver and bronze... but antoniniani are tricky.

Quality¶

Obviously, quality of coins varies from slugs to EF. All listings are EF+ and photophopped to the max. So I did not text mine for quality.
I am mighty tempted to do a Hugh transform on circles in figures, de-prospective them, pump the contrast and measure the difference from Gaussian noise. The reason I'd like to do that is because a deskewed and contrasted image is the first, hardest and most crucial step in making a machine learning for coin classifier.

fakes¶

And there are the fake sellers. Unfortunately, seller data is not given by a normal search. Otherwise I'd have crosschecked with the list of fake sellers.

# load api
api = Finding(config_file='eBay.yaml')

def get_page(page, keywords=None, categoryId=None):
    assert keywords or categoryId, 'No keywords or categoryId'
    #findItemsAdvanced findCompletedItems'
    if keywords:
        query = '<keywords>{keywords}</keywords><paginationInput><pageNumber>{page}</pageNumber></paginationInput>'.format(keywords=keywords, page=page)
    else:
        query = '<categoryId>{categoryId}</categoryId><paginationInput><pageNumber>{page}</pageNumber></paginationInput>'.format(categoryId=categoryId, page=page)
    try:
        response = api.execute('findCompletedItems',query)
    except ConnectionError as e:
        print(e.response.dict())
        raise e
    return response.dict()
    
def get_all_pages(**search):
    '''
    search={'keywords': 'xxx'} or {'categoryId': 'xxx'}
    '''
    data = get_page(1,**search)
    for page in range(2,min(int(data['paginationOutput']['totalPages'])+1,101)):
        newdata = get_page(page,**search)
        if 'searchResult' not in newdata:
            print('****ODD****')
            print(newdata)
        else:
            data['searchResult']['item'].extend(newdata['searchResult']['item'])
            data['searchResult']['_count']=int(newdata['searchResult']['_count'])+int(data['searchResult']['_count'])
            data['paginationOutput']['pageNumber']+='+'+newdata['paginationOutput']['pageNumber']
    return data

# consolidate by emperor
from collections import OrderedDict
by_emp = OrderedDict([(emp['name'],[]) for emp in emperors])
by_emp['Unknown'] = []
for item in alldata['searchResult']['item']:
    if 'bidCount' in item['sellingStatus']:
        if item['sellingStatus']['bidCount'] == 0:
            continue
    else:
        continue
    assert item['sellingStatus']['convertedCurrentPrice']['_currencyId'] == 'USD', item['sellingStatus']['convertedCurrentPrice']['_currencyId']
    title = item['title']
    price = item['sellingStatus']['convertedCurrentPrice']['value']
    candidates = [emp for emp in emperors if emp['name'].lower() in title.lower()]
    if candidates:
        emperor = sorted(candidates, key=lambda x: len(x['name']), reverse = True)
        name = emperor[0]['name']
        
    else:
        name = 'Unknown'
    cand_coin = [c.lower() for c in denominations if c.lower() in title.lower()]
    if cand_coin:
        coin = sorted(cand_coin, key=lambda x: len(x), reverse = True)[0]
    else:
        cand_coin = [c.lower() for c in ('gold','silver', 'bronze', 'copper','billion') if c.lower() in title.lower()]
        if cand_coin:
            coin = cand_coin[0]
        else:
            coin = 'unknown'
    by_emp[name].append({'price': price, 'coin': coin, 'title': title})

x = 0.
y = 0.
width = 100.
height = 100.

values = [len(by_coin[coin]) for coin in by_coin if coin not in ('unknown','gold','bronze', 'silver', 'copper')]
total = sum([len(by_coin[coin]) for coin in by_coin])
names= list([coin.capitalize() for coin in by_coin if coin not in ('unknown','gold','bronze', 'silver', 'copper')])

normed = squarify.normalize_sizes(values, width, height)
rects = squarify.squarify(normed, x, y, width, height)

# Choose colors from http://colorbrewer2.org/ under "Export"
color_brewer = ['rgb(166,206,227)','rgb(31,120,180)','rgb(178,223,138)','rgb(51,160,44)','rgb(251,154,153)','rgb(227,26,28)','rgb(253,191,111)','rgb(255,127,0)','rgb(202,178,214)','rgb(106,61,154)','rgb(255,255,153)']
shapes = []
annotations = []
counter = 0
anno_i = 0

for r in rects:
    shapes.append( 
        dict(
            type = 'rect', 
            x0 = r['x'], 
            y0 = r['y'], 
            x1 = r['x']+r['dx'], 
            y1 = r['y']+r['dy'],
            line = dict( width = 2 ),
            fillcolor = color_brewer[counter]
        ) 
    )
    annotations.append(
        dict(
            x = r['x']+(r['dx']/2),
            y = r['y']+(r['dy']/2),
            text = names[anno_i],
            showarrow = False
        )
    )
    counter = counter + 1
    anno_i = anno_i + 1
    if counter >= len(color_brewer):
        counter = 0

# For hover text
cointrace = go.Scatter(
    x = [ r['x']+(r['dx']/2) for r in rects ], 
    y = [ r['y']+(r['dy']/2) for r in rects ],
    text = [ str(int(v/total*1000)/10)+'%' for v in values ], 
    mode = 'text',
)
